## OmniCode
A lightweight, extensible IDE
It use wails, deno(preact + tailwindcss + vite) and monaco-editor to build the UI.

## Features
Planned to be a full-fledged IDE for the following languages:
- Web
- JavaScript
- TypeScript
- Java
- Python
- C
- C#
- C++
- Go
- Rust
- Godot
